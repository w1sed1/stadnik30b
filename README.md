## 📋 Опис проєкту
DRIVEKNU — це повноцінний веб-додаток для управління автопарком та бронювання автомобілів. Проєкт реалізовано з використанням архітектури клієнт-сервер та запаковано в контейнери для зручного розгортання.

## 🛠 Технологічний стек
* **Backend:** Python 3.11, Django
* **База даних:** PostgreSQL
* **Frontend:** HTML, CSS, JavaScript, Chart.js (візуалізація статистики)
* **Інфраструктура:** Docker, Docker Compose, Nginx (роздача статики)

## ✨ Основний функціонал
1. **Каталог авто:** Перегляд доступних автомобілів із детальними характеристиками (двигун, коробка, ціна тощо).
2. **Система бронювання:** Можливість оренди авто на обрані дати з вбудованою перевіркою накладок (щоб одну машину не забронювали двічі на той самий день).
3. **Календар завантаженості:** Візуальне відображення зайнятих дат та активних бронювань у вигляді графіка/календаря.
4. **AI-асистент:** Вбудований чат-бот для швидкої підтримки клієнтів.
5. **Автентифікація:** Реєстрація, вхід та безпечне управління сесіями користувачів.
6. **Адмін-панель:** Повне управління даними сайту (додавання/видалення авто, керування бронюваннями).
